📖 How to Use

1. First time setup:

Run python -m tr_to_finary.cli --setup to configure credentials.

2. Sync via Web UI:

Go to Sync → Upload CSV or Fetch from TR → Preview → Sync to Finary.

3. Sync via CLI:

python -m tr_to_finary.cli --fetch --execute

4. View transactions:

After importing, go to Transactions to browse all parsed data.
